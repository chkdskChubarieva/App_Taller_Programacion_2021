Instrucciones para instalar la aplicación
---------------------------------------------------------------------------------------------------------
Ejecutar la aplicación .exe como administrador.

Al momento de instalar la aplicación se debe seleccionar una ruta diferente a la del disco en el que está
instalado el Sistema Operativo, ya que en esta ruta el sistema solo permite la lectura de archivos y el
programa para funcionar necesita modificar archivos, se debe dar permisos como administrador.
